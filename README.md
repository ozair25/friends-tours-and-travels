# ✈️ Friends Tours & Travels

A premium, responsive travel agency website built using **Next.js 15**, **TypeScript**, and **Tailwind CSS**. Designed with a luxury heritage-inspired aesthetic, the website showcases domestic and international tour packages while providing an elegant user experience across all devices.

---

## 🌐 Overview

Friends Tours & Travels is a modern travel agency website focused on delivering a premium visual experience with high performance and SEO optimization.

The website is designed to make it easy for visitors to explore destinations, enquire about tour packages, and contact the agency through WhatsApp.

---

## ✨ Features

- Premium travel-themed UI
- Fully responsive (Mobile, Tablet & Desktop)
- SEO Optimized
- Fast page navigation
- Sticky transparent navigation
- Elegant hero section
- Tour Packages
- Destination Gallery
- Testimonials
- About Us
- Contact via WhatsApp

## 🚀 Getting Started

### Clone the repository

```bash
git clone <repository-url>
```

### Navigate into the project

```bash
cd friends
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```

---

## 📦 Production Build

Build the application

```bash
npm run build
```

Run the production server

```bash
npm run start
```
